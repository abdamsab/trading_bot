## Phase 1: The Intelligence Hub (FastAPI + LLM)

This component runs 24/7 on a cheap cloud Linux VPS (like DigitalOcean or Render) to gather all incoming raw text data, structural alerts, and chart conditions. It strips out the noise and feeds a concise prompt to the LLM.

## 1. The Multi-Modal Prompt Engine

Instead of sending raw text blindly, you must convert news, technical data, and alerts into a structured string or a JSON object.



# app/agent.pyimport osfrom openai import OpenAI

client = OpenAI(api_key=os.environ.get("OPENAI_API_KEY"))

def consult_llm_agent(news_text, chart_data, alert_payload):

    system_prompt = (

        "You are an expert financial risk agent executing trades on Exness. "

        "Analyze the provided News, Chart status, and Technical Alerts. "

        "Respond strictly in valid JSON with keys: 'action' (BUY, SELL, or HOLD), "

        "'symbol' (e.g., EURUSD), 'volume' (float lot size), and 'reason' (brief text)."

    )

    

    user_prompt = f"""

    [BREAKING NEWS]: {news_text}

    [CHART METRICS]: {chart_data}

    [EXTERNAL ALERT]: {alert_payload}

    """

    

    response = client.chat.completions.create(

        model="gpt-4o-mini", # Cost-effective and fast for structural JSON processing

        messages=[

            {"role": "system", "content": system_prompt},

            {"role": "user", "content": user_prompt}

        ],

        response_format={"type": "json_object"}

    )

    return response.choices[0].message.content



## 2. The Webhook Ingestion API

You will expose an endpoint so that external systems (like TradingView alerts or automated news feeds) can instantly ping your AI agent.



# app/main.pyfrom fastapi import FastAPI, BackgroundTasksimport requestsimport jsonfrom .agent import consult_llm_agent

app = FastAPI()

MT5_WINDOWS_GATEWAY_URL = "http://YOUR_WINDOWS_VPS_IP:8000/trade"

def process_and_route(news, chart, alert):

    # Get structured decision from LLM

    llm_decision_json = consult_llm_agent(news, chart, alert)

    decision = json.loads(llm_decision_json)

    

    # If the LLM decides to take action, forward it to the Exness MT5 execution gateway

    if decision.get("action") in ["BUY", "SELL"]:

        try:

            requests.post(MT5_WINDOWS_GATEWAY_URL, json=decision, timeout=5)

        except Exception as e:

            print(f"Failed to route order to MT5 Gateway: {e}")



@app.post("/webhook/alert")async def handle_incoming_signal(data: dict, background_tasks: BackgroundTasks):

    # Extract properties from your webhook source

    news = data.get("news", "No major breaking news items.")

    chart = data.get("chart", "RSI: Neutral, Price above 200 EMA")

    alert = data.get("alert", "TradingView Trigger")

    

    # Process asynchronously to respond to webhook sources instantly

    background_tasks.add_task(process_and_route, news, chart, alert)

    return {"status": "received"}
